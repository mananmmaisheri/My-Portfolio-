## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/mananmmaisheri/My-Portfolio.git
```

### 2️⃣ Navigate into the project

```bash
cd My-Portfolio
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```
